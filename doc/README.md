## 🎯 项目简介

**LinguaFlow** 是一个创新的社交化英语学习平台，结合了：
- **X (Twitter)** 的 Feed 流设计
- **小说阅读** + **游戏化闯关** 的学习方式
- **Apple** 的优雅设计语言
- **Supabase** 和 **Lobe Chat** 的架构最佳实践

## ✨ 核心特性

- 📱 **Feed流系统**: 类X的社交信息流
- 📚 **小说学习**: 电子书阅读器
- 🎮 **闯关系统**: 多样化英语挑战
- 🤖 **AI评分**: 智能评分和反馈
- 💬 **社交互动**: 点赞、评论、分享
- ⚡ **实时更新**: 多用户实时同步

## 🚀 快速开始

```bash
# 1. 安装依赖
pnpm install

# 2. 启动开发服务器
pnpm dev

# 3. 访问应用
# 打开浏览器访问 http://localhost:3000
```

## 🛠️ 技术栈

### 前端
- **Next.js 15** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式系统
- **Framer Motion** - 动画库
- **Zustand** - 状态管理
- **React Query** - 数据查询

### 后端
- **Supabase** - BaaS 平台
- **PostgreSQL** - 数据库
- **Realtime** - 实时订阅
- **Storage** - 文件存储
- **Auth** - 用户认证
