## 🚀 快速开始

1. **启动开发服务器**
   ```bash
   npm run dev
   ```

2. **访问应用**
   - 打开 http://localhost:3000
   - 找到顶部的Quest卡片

3. **体验流程**
   - 点击"Start Reading"开始阅读
   - 尝试不同的主题和字体
   - 滚动到底部完成阅读
   - 点击"Start Challenge"答题
   - 查看成绩和统计
